# cloud-security-journey

GROK overview:
Must-Learn Order (Focus on these first)

Linux Fundamentals
Commands, file system, permissions (chmod, chown), Bash basics, package management (apt).
Why first: Everything in cyber & cloud runs on Linux.

Networking Fundamentals
OSI/TCP-IP model, IP addressing & subnetting, ports, protocols (HTTP, DNS, DHCP), firewalls basics.
Why: Core for security and cloud (VPC, security groups).

Security Basics
CIA triad, threats & attacks, encryption, authentication, OWASP Top 10, basic incident response.
Start with Google Cybersecurity Certificate + TryHackMe Pre Security path.

AWS Cloud Foundations
IAM, EC2, S3, VPC, Security Groups, basic cloud security.
Goal: Finish AWS Cloud Practitioner free training.

Python Scripting (Basics)
Variables, loops, functions, file handling, simple scripts (port scanner, automation).
Start light after Linux — very useful for both cyber and cloud.
